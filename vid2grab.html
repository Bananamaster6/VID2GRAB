Get it into GRAB

  1. Download the JSON above.
  2. Open the JSON Level Editor at grabvr.tools.
  3. Drag the downloaded .json file straight onto the editor — it loads as your level, ready to inspect or tweak.
  4. Export it as a .level file from there, or publish it from inside GRAB.
Prefer a .level file directly? Use the Level ⇄ JSON tool and drop this file in instead.
VID2GRAB is an independent, unofficial tool. Not affiliated with or endorsed by grabvr.tools.
